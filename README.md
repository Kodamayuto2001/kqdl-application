# kqdl-application

## rootProject (kqdl-application)
```
./gradlew composeUp
./gradlew composeDown
```

## java-application (kqdl-resort-reservation-service)
```
./gradlew build
./gradlew run
```

## vue-application (kqdl-resort-service-web-ui)
```
事前にcomposeUpしておく
コンテナに入ってcd app && npm run serveをするか3分程度待つ
http://localhost:8080/にアクセスする。
```
